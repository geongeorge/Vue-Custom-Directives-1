# Custom Directives example

~Highlight directive

![alt text](GIF.gif)

## Directive Use
```
<p v-highlight="'red'">Hello</p>

<p v-highlight.blink="'green'">Hello</p>

<span v-highlight="'#fff'">Hello</span>
```
